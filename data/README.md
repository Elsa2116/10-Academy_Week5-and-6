# Data Directory

This folder stores the project data layout.

- `raw/` contains the original input CSV files required by the scripts.
- `processed/` contains cleaned, feature-engineered, and resampled outputs generated by the pipeline.

The raw source files are not committed to git. Add these files locally before running the workflow:

- `data/raw/Fraud_Data.csv`
- `data/raw/IpAddress_to_Country.csv`
- `data/raw/creditcard.csv`
