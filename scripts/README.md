# Scripts

- `run_task1.py`: cleans, enriches, engineers features, produces EDA plots, and documents class imbalance/resampling.
- `train_models.py`: trains Logistic Regression and Random Forest models for one or both datasets.
- `run_shap.py`: generates SHAP summaries for the saved best model.
- `generate_final_report.py`: assembles the final markdown report from the saved metrics and SHAP artifacts.

Run from the repository root after installing requirements.
